## 🍷 Wine Intelligence Elite (V3.0)

**Smart Management for Premium Wine Collections**

Μια λιτή και αποδοτική εφαρμογή για τη διαχείριση και αξιολόγηση βραβευμένων ετικετών κρασιού. 
Σχεδιασμένη με έμφαση στο UI/UX και τη χρηστικότητα, η εφαρμογή βοηθά στη λήψη γρήγορων 
αποφάσεων αγοράς μέσω υπολογισμού Value for Money (VfM).
$$VfM\_Score = \frac{Rating}{Price} \times 10$$

---

## 🚀 Βασικές Λειτουργίες
* **VfM Index:** Αυτόματος υπολογισμός της σχέσης ποιότητας-τιμής.
* **Smart Search:** Αναζήτηση ανά ποικιλία, τύπο ή περιοχή.
* **Market Sync:** Άμεση σύνδεση με το Skroutz για έλεγχο τιμών σε πραγματικό χρόνο.
* **Management Tools:** Επεξεργασία δεδομένων, προσθήκη σημειώσεων και εξαγωγή σε Excel.

---

## 📖 Οδηγός Χρήσης (User Guide)
1. **Αναζήτηση:** Χρησιμοποιήστε το Search Bar για να βρείτε το κρασί που σας ενδιαφέρει.
2. **Αξιολόγηση:** Ελέγξτε το **VfM Index**. Όσο υψηλότερος είναι, τόσο καλύτερη η σχέση τιμής/βαθμολογίας.
3. **Έλεγχος Τιμής:** Πατήστε το link "Link" στη στήλη Skroutz για να δείτε την τρέχουσα τιμή αγοράς.
4. **Αποθήκευση:** Μετά από κάθε αλλαγή στον πίνακα, πατήστε το κουμπί **💾 ΑΠΟΘΗΚΕΥΣΗ**.
